Advanced Image Processing Filters

This app allows users to apply various image processing filters to images uploaded through a web interface. The filters include techniques like blurring, grayscale conversion, edge detection, and more. The app uses Streamlit for the UI and OpenCV for image processing.

Features:

- Apply various filters including Blur, Grayscale, Thresholding, Erosion, Dilation, and more.
- View histograms of the original and processed images side by side.
- Interactive controls for adjusting filter parameters.

Installation:

1. Clone this repository to your local machine:
    git clone https://github.com/username/repository-name.git
2. Navigate to the project folder:
    cd repository-name
3. Install the required dependencies:
    pip install -r requirements.txt

Usage:

1. Run the app:
    streamlit run app.py
2. Upload an image and select a filter from the dropdown.
3. Adjust the parameters to see the changes in real-time.

Dependencies:

- Python 3.x
- Streamlit
- OpenCV
- NumPy
- Pillow
- Matplotlib
